# koda
Inicio de proyecto final "Koda" (app) es una aplicación móvil que genera rutinas de ejercicio
y planes de alimentación personalizados usando inteligencia artificial.

La aplicación permite a los usuarios registrarse, ingresar sus datos
físicos y objetivos (bajar peso, ganar masa muscular, mejorar condición),
y recibir notificaciones automáticas adaptadas a su perfil.

Tecnologías usadas
Backend: Django
API: Django REST Framework
App móvil: Flutter
Base de datos: MySQL
Lenguaje: Python

Funcionalidades
Registro de usuarios
Inicio de sesión
Generación de rutinas personalizadas
Plan de alimentación recomendado
Seguimiento de progreso
Recomendaciones usando IA

Arquitectura del sistema
Flutter (App móvil)
API REST Django (Backend)
MySQL (Base de datos)
